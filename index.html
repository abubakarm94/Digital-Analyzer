The overall purpose of our project is to convert handwriting characters into digital text.

Introduction

The goal of our project is to convert handwritten characters into digital text. The bulk of the work is done through a neural network that allows the computer to learn various samples from persistent training. In doing this, we are trying to figure out the best way to implement a neural network that would work analogous with programs dealing with image thresholding and connected components. Due to the time constraint ahead of us, we are also trying to figure out the fastest way to implement this. We chose this project because originally we wanted to do something with deep learning. We were very fascinated with the way that various companies such as Facebook have been able to do things concerning this topic and thus decided to do something similar to this.

Background

For most humans recognizing handwritten characters and digits seems like a fairly easy task to do. However, this simple task becomes very difficult when trying to request a computer to recognize handwritten characters and digits. What once seemed as an easy task becomes extremely difficult. As we have learned through online research and through speaking with Dr. Kim, we have learned that Neural Networks have tried to alleviate the difficulty of handwriting recognition. What Neural Networks do is uses large numbers of handwritten samples to infer rules for recognizing digits. While researching there have been many developments of digit recognition. Various methods of digit recognition have utilized a small sample of handwritten digits which have decreased the accuracy of the program. When using these sample programs there have been many times were the program has incorrectly guessed the digit that was written. For our project we will try to implement digit recognition first by using the MNIST database of handwritten digits which has a training set of 60,000 examples to try to ensure that out neural network is as accurate as possible. Given our time constraint for this semester if achieving digit recognition is feasible, we will then attempt to implement character recognition. Given that there is not a known and reliable database for English characters, our implementation may not be as accurate because we will not have a large sample size however, we will still use a smaller sample size and try to get a decent accuracy rate for both digit and English character recognition.
