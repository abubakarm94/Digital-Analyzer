Digital Analyzer

The overall purpose of our project is to convert handwriting characters into digital text.

View on GitHub Download .zip Download .tar.gz

Introduction

The goal of our project is to convert handwritten characters into digital text. The bulk of the work is done through a neural network that allows the computer to learn various samples from persistent training. In doing this, we are trying to figure out the best way to implement a neural network that would work analogous with programs dealing with image thresholding and connected components. Due to the time constraint ahead of us, we are also trying to figure out the fastest way to implement this. We chose this project because originally we wanted to do something with deep learning. We were very fascinated with the way that various companies such as Facebook have been able to do things concerning this topic and thus decided to do something similar to this.

Background

So far our group has researched Image Thresholding, Connect Component Labeling, and a Neural Networks. After our research we each individually looked up online code to gain a feeling and understand of how these components would be implemented. After finding online code, we then each attempted to get the code running on our laptops, we verified with other members of the group if we were having trouble getting the code to work and now we have worked out all issues and now we currently have code running for each component. Now that we have some understanding we will now look into how we can tailor each of these individual components so that they can work in unison so that handwriting recognition can be implemented. So currently we will be looking at the most optimal solution as to how was can use the code we found online so that all components can work together.
